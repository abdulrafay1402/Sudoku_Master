How to Play

  • Each row must contain all numbers from 1 to N (where N is grid size) without repetition
  • Each column must contain all numbers from 1 to N without repetition
  • Each subgrid (√N x √N) must contain all numbers from 1 to N without repetition
  • Click on a cell to select it, then type a number to fill it
  • Press Delete or Backspace to clear a cell
